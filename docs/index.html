Settings

Settings page.

 

Trip Detail

The detail of the flights on a trip. 
  • Here, the user can add adittional flights. 
  • Long-tapping a flight will delete it.

  • Clicking the airplane icon on the toolbar opens a flight detail window

  • Long-tapping a flight opens a dialog where the flight can be deleted

 

Trip Detail - Expanded header

The detail of the flights on a trip. Here, the user can add extra flights, save the trip as-is or discard the whole process.

 

Trip detail 1

This page displays a life view of a flight if it is currently ongoing. Otherwise it just shows the route from the destination airport to the arrival airport.
  • The page can be swiped left to view the next flight in the trip.
  • The card can be swiped down for a zommed-in view of the map

 

Trip detail 2

This page displays a life view of a flight if it is currently ongoing. Otherwise it just shows the route from the destination airport to the arrival airport.
  • The page can be swiped left or right to view the next or previous flight in the trip.
  • The card can be swiped down for a zommed-in view of the map




 

Trip detail 3

This page displays a life view of a flight if it is currently ongoing. Otherwise it just shows the route from the destination airport to the arrival airport.
  • The page can be swiped right to view the previous flight in the trip.
  • The card can be swiped down for a zommed-in view of the map

 

Add a flight

This page allows the user to add a flight to a trip.
Typing the search field looks up Airlines and Airports

 

AAF - Search

If the user selects an Airport, the app will search flights by route. 
Otherwise, if the user selects an Airline, the app will search by flight #.

 

AAF - By route 1

User searches or taps on a recent destination airport, or taps the departure airport to change it.

 

AAF - By route 2

The user selects the date of the flight and taps search. Or can tap on the departure or destination airports to change them.

 

AAF - By route 3

The app displays a list of flights by the entered route criteria. Here, the user can select a flight to add to the trip. The departure, arrival and flight date can be edited. 

 

AAF - By flight num 1

User types in the flight number, clicks search button

 

AAF - By flight num 2

User selects a date for the flight, taps on search button

 

AAF - By flight num 3

If a flight exists, it will appear here, otherwise an error message will show
