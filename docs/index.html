Add a flight

Typing the search field looks up Airlines and Airports

 

AAF - Search

If the user selects an Airport, the app will search flights by route. Otherwise, if the user selects an Airline, the app will search by flight #.

 

AAF - By route 1

User searches or taps on a recent destination airport.

 

AAF - By route 2

The user selects the date of the flight and taps search

 

AAF - By route 3

The app displays a list of flights by the entered route criteria. If the user made a mistake, he/she can change the departure, arrival or flight date. Otherwise select a flight to add to the trip.

 

AAF - By flight num 1

User types in the flight number, clicks search button

 

AAF - By flight num 2

User selects a date for the flight, taps on search button

 

AAF - By flight num 3

If a flight exists, it will appear here, otherwise an error message will show

 

AAF - Trip Detail

The detail of the flights on a trip. Here, the user can add extra flights, save the trip as-is or discard the whole process.
